⚙️ Первый запуск — подключите базу данных

Чтобы все устройства работали с одними данными, нужно подключить Supabase (бесплатно). Займёт ~10 минут.

Шаг 1. Зарегистрируйтесь на supabase.com → New Project → придумайте название и пароль.
Шаг 2. В проекте откройте SQL Editor → вставьте и выполните этот SQL:

create table daily_entries (id bigint generated always as identity primary key, entry_date date not null unique, data jsonb not null, updated_at timestamptz default now()); create table hot_clients (id bigint generated always as identity primary key, data jsonb not null, created_at timestamptz default now()); alter table daily_entries enable row level security; alter table hot_clients enable row level security; create policy "allow all" on daily_entries for all using (true) with check (true); create policy "allow all" on hot_clients for all using (true) with check (true);
Шаг 3. Откройте Settings → API. Скопируйте Project URL и anon public key и вставьте сюда:
